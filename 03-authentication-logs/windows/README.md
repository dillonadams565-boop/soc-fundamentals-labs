# Lab 3 — Windows Authentication Log Review

## Objective
Review Windows Security logs to identify failed and successful authentication events and understand what normal and abnormal logon activity looks like.

## Environment
- Operating System: Windows
- Tool: Event Viewer
- Log Source: Windows Security Log

## Scope
This lab focuses on local authentication activity only. Controlled failed logon attempts and successful logons will be generated and reviewed.

## Events of Interest
- Event ID 4625 — Failed logon
- Event ID 4624 — Successful logon

## Status
Lab in progress. Findings and evidence will be added after completing log generation and analysis.
